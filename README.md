### Hi, I'm Izzah!


I hold an education background in economics and data science. I'm an 
agri-commodities analyst turned data-driven explorer, currently interested in 
economics, the interconnectedness of markets, and data analysis. I enjoy building 
machine learning models and exploring in my free time.

**What I work with:**
- Python for predictive modeling & explainability
- Databricks for large-scale data pipelines and cross-correlation analysis

**Featured projects:**
- **Company Bankruptcy Prediction** — Full EDA-to-dashboard ML pipeline (Taiwan Economic Journal dataset), built in Colab + Looker Studio
- **House Price Prediction in Kuala Lumpur** — End-to-end pipeline predicting 
  property prices, enriched with nearby amenities (train stations, schools, malls) 
  via Google APIs; compared Linear Regression, Random Forest, and Gradient Boosting
- **Sentiment Analysis of Agoda Hotel Reviews** — Big data pipeline on Azure 
  (Medallion Architecture) classifying app reviews at scale, with Power BI dashboards; 
  best model (Linear SVM) hit 87–90% accuracy
